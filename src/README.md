# WazobiaTax.ng - Tax Compliance PWA for Nigerian MSMEs

A comprehensive Progressive Web App designed to simplify tax compliance for Nigeria's nano, micro, and small businesses (MSMEs) in accordance with the Nigeria Tax Administration Act, 2025 (NTAA).

- Market vendors

## ‚ú® Key Features

### Core Functionality
- **TIN Registration**: Simplified registration using BVN/NIN with auto-fill
- **Digital Ledger**: Track income and expenses with OCR receipt scanning
- **Automated Tax Returns**: File VAT, PAYE, and income tax returns
- **Payment Integration**: Secure payments via Paystack/Flutterwave
- **Tax Clearance Requests**: Generate digital Tax Clearance Certificates
- **Compliance Dashboard**: Real-time compliance score and deadline tracking

### Accessibility Features
- **Multilingual Support**: English, Pidgin, Hausa, Yoruba, Igbo
- **Voice Activation**: Hands-free entry and navigation
- **OCR Receipt Scanning**: Camera-based receipt capture
- **Offline-First**: Works without internet with sync capability
- **WCAG 2.1 Compliant**: High contrast ratios, large tap targets

### User Experience
- Mobile-first design (390x844px iPhone 14 frame)
- Clean sans-serif typography (16px base)
- Color-coded status indicators
- Micro-interactions and animations
- Glassmorphism and subtle gradients
